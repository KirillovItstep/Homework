Задача 1. 
Вариант 1. Дан массив n целых чисел, содержащий не менее двух нулей. 
Найти сумму чисел, расположенных между последними двумя нулями.
Вариант 2. Дан массив n целых чисел. Найти длину самой длинной возрастающей последовательности.
Вариант 3. Дан массив n целых чисел. Найти элемент, который встречается в массиве чаще всего.

Задача 2.
Вариант 1. Дан символ и строка, состоящая из произвольных символов. Удвоить вхождение данного символа в строку.
Вариант 2. Дана строка, состоящая из произвольных символов. Найти в ней первый неповторяющийся символ.
Вариант 3. Дана строка, состоящая из произвольных символов. Найти символ, который встречается в ней  чаще всего.

Задача 2.
Вариант 1. Дан двумерный массив nxn целых чисел. Найти минимальный из максимальных элементов ее строк.
Вариант 2. Дан двумерный массив nxn целых чисел. Выполнить его поворот против часовой стрелки.
Вариант 3. Дан двумерный массив nxn целых чисел. Вывести строку, сумма элементов которой минимальна.
